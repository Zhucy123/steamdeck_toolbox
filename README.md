# SteamDeck 工具箱

一个专为 SteamDeck 打造的一站式系统管理与增强工具，集成系统优化、软件安装、网络加速、双系统支持、游戏增强等 **31 项**实用功能，全面覆盖日常使用与进阶调校需求。

> **版本：v1.3.2** | 更新日期：2026-06-30  
> 制作人：薯条 & DeepSeek

## 🚀 一键安装

在 Konsole 中执行以下命令即可安装：

curl -L https://shoturl.cn/n2eFhl | sh

## ✨ 特性亮点

- **智能系统检测**：自动识别单 / 双系统，部分功能根据系统类型动态显示  
- **引导管理**：支持 Clover 图形化引导的安装、重装、修复（单/双系统均可用）  
- **双系统专属支持**：引导修复（Clover 启动顺序）、互通盘配置仅对双系统用户开放  
- **一键软件管理**：涵盖办公、通讯、浏览器、云盘、音乐等常用应用  
- **深度性能优化**：虚拟内存调整（CryoByte33 方案 / ZRAM / 传统 Swap）、Steam 缓存管理  
- **游戏增强组件**：摇杆校准、LSFG‑VK / Framegen 插件、小黄鸭软件（Lossless Scaling）  
- **持续更新**：内置自动更新，无缝获取最新功能  

## 📋 功能列表

### 系统优化与修复
- 配置国内源（Flatpak 镜像加速）  
- 虚拟内存调整（CryoByte33 推荐 / Swap 文件 / ZRAM / 清理所有配置）  
- 修复磁盘写入错误  
- 清理 hosts 缓存  
- 设置管理员密码  
- **切换显示服务器（X11 / Wayland）** – 远程连接异常时一键切换图形后端  

### 引导管理（单 / 双系统通用）
- 安装 / 重装 Clover 图形化引导管理器  

### 网络与远程工具
- UU 加速器插件  
- 迅游加速器插件  
- ToMoon 插件（安装 / 卸载）  
- AnyDesk 远程协助  
- ToDesk 远程协助  
- Steam 社区 302 访问工具  

### 常用软件安装
- WPS Office  
- QQ  
- 微信  
- QQ 音乐  
- 百度网盘  
- Microsoft Edge 浏览器  
- Google Chrome 浏览器  

### 游戏增强与 Decky 插件
- 校准摇杆  
- 插件商店（Decky Loader）安装 / 卸载  
- LSFG‑VK 插件（安装 / 更新 / 卸载）  
- Framegen 插件（安装 / 更新 / 卸载）  
- 小黄鸭软件（Lossless Scaling，帧生成工具）  
- Steam 缓存管理器（图形界面，支持删除 / 移动着色器缓存与兼容数据）  

### 双系统支持（仅双系统用户可见）
- 修复引导（Clover 启动顺序自动设置）  
- 修复互通盘（Windows – SteamOS 数据共享）  

### 应用管理与维护
- 更新已安装应用（单个或全部）  
- 卸载已安装应用  
- 工具箱自动更新（启动时检测）
